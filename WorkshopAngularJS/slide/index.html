Perché dovrei usare AngularJS?

E' un framework pensato per le Web Apps (non per i siti) e sopratutto per le Single Page Web Apps.

Si basa sul concetto di data-binding che velocizza molti passaggi.

Utilizza un design pattern MVC/MVVM

Può essere usato in progetti interi o anche solo in piccole porzioni.
